## Focus

- **Problems people are actually experiencing** — outages, bugs, regressions, pricing pain, broken promises
- **Controversies** — lawsuits, ethical issues, copyright disputes, safety concerns
- **Backlash & frustration** — community complaints, viral rants, mass migrations
- **Failures** — tools shutting down, features removed, quality degradation
- **Trust issues** — data privacy, hallucinations, reliability concerns
